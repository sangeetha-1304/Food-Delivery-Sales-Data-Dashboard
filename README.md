# Food-Delivery-Sales-Data-Dashboard
## Objective: 
To analyze and visualize key food delivery performance metrics such as total sales, average order value, delivery efficiency, and customer demographics.

## Data Cleaning: 
We transformed and cleaned the dataset, including handling missing values, correcting data types, and merging multiple data sources.

## Data Analysis: 
DAX measures were created for key performance indicators such as total sales, sales by category, average order value, and on-time delivery percentage.

## Data Visualization: 
Various Power BI visualizations such as card visuals, donut charts, and stacked column charts were used to represent key metrics and trends.

## Insights: 
Key findings from the analysis include the impact of food category and delivery types on sales, as well as customer age group trends.

## Deliverable: 
An interactive Power BI report visualizing sales performance, delivery metrics, and customer demographics to drive operational improvements.
